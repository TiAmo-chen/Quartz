# 1. 相对路径 ，用正斜杠’/‘

## 1.1 向上一级目录指向
./    表示当前路径，相当于 E:\test\file<br />../  表示当前路径的上一级路径，相当于 E:\test<br />../../  表示当前路径上上一级路径，相当于 E:<br />更多指向上级路径的表示以此类推。

## 1.2 向下一级目录指向
./data  表示当前路径下一级路径，相当于 E:\test\file\data<br />./data/xxx   表示当前路径的下下一级路径，相当于 E:\test\file\data\xxx<br />更多向下指向的路径的表示以此类推。


# 2. 绝对路径，用反斜杠 '\'
E:\test<br />E:\test\file<br />E:\test\file\data


# 3.C\C++ 编程中的相对路径和绝对路径使用
C 语言中，反斜杠’\‘表示转义字符，所以绝对路径需要如下表示<br />FILE * fp;<br />fp = fopen(**"E:\\test\\file\\data\\d.txt"**,"r");

也可以用相对路径表示，不受转义字符限制：<br />FILE * fp;<br />fp = fopen(**"E:/test/file/data/d.txt"**,"r");

或者，当前路径 E:\test\file 下，相对路径表示为<br />FILE * fp;<br />fp = fopen(**"./data/d.txt"**,"r");
